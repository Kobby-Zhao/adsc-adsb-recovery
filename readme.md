# 面向跨洋稀疏 ADS-C 轨迹恢复的研究

## 研究背景

跨洋航班的 ADS-C 轨迹通常具有采样稀疏、时间间隔长、连续真值难获取等特点，这使得轨迹恢复问题明显区别于高频 ADS-B 场景。尤其在长时间缺失区间内，模型不仅需要补全空间位置，还需要尽量保持高度变化的物理合理性、边界连续性和整体时序一致性。因此，本项目聚焦于跨洋稀疏监视条件下的三维轨迹恢复，重点研究长 gap 条件下的高度恢复与整体轨迹重建方法。

## 方案简介

本项目采用面向稀疏轨迹恢复的分层建模思路，以双向时序编码为主干，在利用缺失区间两侧锚点信息的基础上，对 gap 内部轨迹进行恢复。整体方法重点关注高度维度在稀疏条件下的特殊性，将其作为独立研究对象，与平面轨迹恢复协同建模，以提升长缺口场景下的稳定性与可解释性。围绕这一目标，项目进一步包含课程学习、阶段化数据构造、消融实验和真实场景对比等模块，用于持续评估不同恢复策略在跨洋 ADS-C 场景中的适用性。
